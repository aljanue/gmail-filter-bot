Maintenance Manual

Gmail Security Bot - Google Cloud Platform

1. Status and Monitoring

Use these commands to verify that the bot is operating correctly on your Debian instance.

Health Check (Status)

sudo systemctl status gmailbot

Real-time Logs

To see who is messaging the bot or which emails are being processed right now:

sudo journalctl -u gmailbot -f

2. Service Management

Execution control for the background process.

Restart

Required after changing the .env file or the source code.

sudo systemctl restart gmailbot

Stop and Start

sudo systemctl stop gmailbot sudo systemctl start gmailbot

3. Configuration and Files

Critical paths within the server.

Project Folder

cd ~/mail_filter_bot

Edit Environment Variables

nano ~/mail_filter_bot/.env

Consult Email Database

cat ~/mail_filter_bot/config.json

4. Code Update

If you make improvements on your Mac and want to upload them:

  • Upload the new bot_produccion.zip via SSH terminal.
  • Execute the following block:
cd ~/mail_filter_bot unzip -o ../bot_produccion.zip sudo systemctl restart gmailbot
